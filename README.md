# Cordinsanity — Revenge Plugins

> Privacy-focused plugins for [Revenge](https://github.com/revenge-mod) / Vendetta (Discord mod)

---

## Warum diese Plugins?

Die meisten Discord-Mods und Plugins speichern Tokens im Klartext, loggen alles mit und haben null Sicherheitsfeatures. Das hier ist anders.

Alle Plugins hier sind von Grund auf mit **Privacy- und Security-Fokus** gebaut:
- Tokens werden **verschlüsselt** gespeichert (AES-256-GCM)
- Kein Klartext in Storage oder Logs
- Netzwerk-Monitoring statt blindes Vertrauen
- Einstellbar — du entscheidest was aktiv ist

---

## Plugins

| Plugin | Beschreibung | Install |
|--------|-------------|---------|
| [MoreAlts](plugins/MoreAlts/README.md) | Sicherer Account-Switcher mit AES-256 Token-Verschlüsselung, PIN-Lock, Biometrie & mehr | [Link](https://cordinsanity.github.io/revenge-plugins/MoreAlts/index.js) |
| [GhostMode](plugins/GhostMode/README.md) | Unsichtbar schalten, Tipp-Indikator & Lesebestätigungen blockieren — Toggle direkt im You Bar | [Link](https://cordinsanity.github.io/revenge-plugins/GhostMode/index.js) |
| [TokenGuard](plugins/TokenGuard/README.md) | Überwacht alle Netzwerk-Anfragen und warnt wenn ein Plugin deinen Token weitersendet | [Link](https://cordinsanity.github.io/revenge-plugins/TokenGuard/index.js) |

## Installation

1. **Revenge** auf Discord öffnen
2. **Settings → Plugins → +**
3. Plugin-URL einfügen (siehe Tabelle oben)
